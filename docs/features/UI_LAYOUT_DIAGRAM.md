## Z-Order (Layering)

From back to front:
1. Star field (3D space)
2. Ship and entities (3D)
3. Panel backgrounds
4. Panel borders
5. Text elements
6. Tooltips (future)

## Spacing and Margins

- Panel padding: 0.02 units
- Panel margin: 0.01 units
- Text line spacing: 0.03 units
- Header height: 0.05 units
- Border thickness: 0.002 units
